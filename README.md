# Fake Job Detection System

## Overview
This project aims to create a robust system that can identify and filter out fake job postings.

## Features
- Job posting analysis
- Machine learning model that predicts job authenticity
- User-friendly interface
- Chrome extension for seamless browsing

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- Database: MongoDB
- Machine Learning: Python, Scikit-learn
- Chrome Extension: JavaScript

## Project Structure
```
fake-job-detection-system/
├── frontend/
├── backend/
├── ml_model/
├── chrome_extension/
└── README.md
```

## Setup Instructions

### Frontend
1. Navigate to the `frontend` directory.
2. Run `npm install` to install dependencies.
3. Start the application using `npm start`.

### Backend
1. Navigate to the `backend` directory.
2. Run `npm install` to install dependencies.
3. Start the server using `npm start`.

### ML Model
1. Navigate to the `ml_model` directory.
2. Install required libraries using `pip install -r requirements.txt`.

### Chrome Extension
1. Navigate to the `chrome_extension` directory.
2. Load the unpacked extension in Chrome by navigating to `chrome://extensions/`.

## Database Setup
1. Install MongoDB on your system.
2. Create a database named `fake_job_detection`.
3. Configure connection settings in the backend.
